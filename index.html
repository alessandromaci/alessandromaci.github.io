My Portfolio

A brief description of some projects I worked on in the last months

mac-insurance

It is a permissionless insurance protocol that can be used to create and request an insurance based on oracle data.

We built an insurance type to protect from ERC20 token price loss for this hackathon by using Chainlink oracle price-feeds. Our vision, however, is not just limited to insurance against price loss. The goal of the protocol is to support as many insurance use-cases as possible based on the oracle data availability.

Punk Cities

Collaborative play-to-earn game based on metadata from places in cities

Players will play as Solarpunks or Cyberpunks and choose a hometown where they will be rewarded with Energy or Chips, by registering and verifying parks, landmarks, universities, bike trails, industrial zones, and other elements from their hometowns as NFTs. These rewards can be used to upgrade their hometown places, increase their rewards for all verifiers, and mark the place for the team with the most verifications.

  • Developed the smart contract as well as the integration between the contract and the UI. Take a look at the Github repo
  • This project entails the use of ERC1155, IPFS, and NFT.Storage
